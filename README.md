# 功能描述
基于ng2创建的一个图片上传组件,支持功能

(1)显示图片批量上传

(2)显示图片上传状态,上传成功后,打钩提示上传完成

(3)上传图片排序(根据日期\文件大小\拍摄时间)

(4)大图预览(大图预览功能)
